Что-бы пользоваться приложением Национального Детского технопарка тебе потребуется скачать приложение Expo Go из AppStore или GooglePlay

Для iOS
1. Нажмите на эту кнопку
2. Нажмите на кнопку "На экран домой"
Для Android
1. Нажмите на эту кнопку
2. Нажмите на кнопку "На экран домой"
